#  Prueba técnica - Weremote

- Clona el repositorio
- Ejecuta en la terminal npm install para instalar dependencias
- Ejecuta en la terminal npm run dev para correr el proyecto 

## Tecnologías y metodologías aplicadas 

- React
- Sass
- JS ES6
- Nomenclatura BEM
- Optimización SEO
